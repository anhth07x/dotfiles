### Auto-Installation:

- #### Clone my repo
```bash
git clone https://github.com/tknightz/dotfiles.git
cd dotfiles
```

- #### Install all configurations
```bash
./install.sh
```

- #### Install only zsh, ohmyzsh, powerlevel9k
```bash
./install_omz.sh
```

- #### Install only nvim configurations
```bash
chmod +x install_vim
./install_vim.sh
```
